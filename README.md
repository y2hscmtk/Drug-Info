# 이게뭐약?(Drug-Info)

이게뭐약?은 API통신 연습을 위한 사이드 프로젝트입니다.

공공데이터 포탈 오픈API [식품의약품안전처_의약품개요정보(e약은요)](https://www.data.go.kr/data/15075057/openapi.do)를 활용하여 제작하였습니다.
